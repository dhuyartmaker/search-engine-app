### Built With

* NodeJS v20 (ExpressJs)
* ReactJs v18
* MongoDB (v4 upper)
* ChakraUI

## Getting Started

This is some step to install
### Installation
* After clone repository
    ```
    npm install
    ```
* Create env file follow sample
    ```
    .env
    ```
* Start
    ```
    npm start
    ```
